Elemen Dasar dalam HTML

| Elemen              | Fungsi                               |
| ------------------- | ------------------------------------ |
| `<h1>` - `<h6>` | Heading atau judul                   |
| `<p>`             | Paragraf                             |
| `<a>`             | Link (Tautan)                        |
| `<img>`           | Menampilkan gambar                   |
| `<ul>` & `<ol>` | List (daftar)                        |
| `<table>`         | Tabel                                |
| `<div>`           | Container untuk pengelompokan elemen |
| `<span>`          | Inline container                     |
